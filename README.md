## Build And Run 

- Build AST Generator

```
make build_gast 
```

- Run AST Generator

```
make run_gast
```

- Build Jlox interpreter

```
make build
```

- Run Jlox interactively

```
make run
```

- Run Jlox file

```
make run src=./tests/file.jl
```
